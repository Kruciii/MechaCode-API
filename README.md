Jak to odpalić? 
1) Musisz mieć vs code, Docker Desktop (z włączonym WSL2)
2) klonujesz repozytorium
3) w terminalu wpisujesz docker-compose up --build
4) Czekasz az wszystko sie poinstaluje
5) Wchodzisz i sprawdzasz czy działa : 
Aplikacja Webowa: http://localhost:5173
Dokumentacja API (Swagger): http://localhost:8000/docs
Sędzia (Judge0): http://localhost:2358/docs
